Preguntas frecuentes

Viajero internacional

¿Qué hago si no tengo servicio de Internet (datos)?

  • Verifica que la itinerancia de datos de tu teléfono (Roaming de datos) esté activada.
  • Resetea el equipo.
  • En caso de ser necesario, cambia la opción de 4G a 3G o 2G.

La configuración de estas funciones puede variar dependiendo del Sistema Operativo, la marca y modelo del teléfono.

¿Qué hago si mi equipo no detecta señal?

  • Opción 1: Apaga y enciende tu equipo, la búsqueda de redes tardará algunos segundos.
  • Opción 2: En el menú “Configuración” o “Ajustes” de tu celular, cambia la selección de red de forma manual.
  • Opción 3: Verifica que el “Modo Avión” se encuentre desactivado.

Los pasos varían dependiendo del Sistema Operativo, la marca y el modelo del teléfono celular.

¿Qué sucede si utilizo Internet a través de WiFi en el extranjero?

Si tienes activa la opción de Wi-Fi y al mismo tiempo itinerancia de datos/Roaming de Datos en tu celular, se pueden generar cargos en tu factura en caso de que pierdas la conexión Wi-Fi y continúes navegando en la red móvil, sin que percibas el cambio.

Si deseas hacer uso exclusivo de WiFi te sugerimos desactivar itinerancia de datos y así evitar cargos adicionales.

¿Cómo debo marcar para hacer una llamada o enviar un SMS desde el extranjero?

  • Para llamar a México desde el extranjero: Fijo: (+) 52 - el código de área - Número Celular: ( + ) 52 - 1 - Código de área -Número
  • La marcación mundial desde el país donde te encuentres: (+) - Código de país - Código de área - Número al que quieres llamar
  • Para enviar SMS ( + ) Código de país - Código de Área - Número
